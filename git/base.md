# Базовые команды гит

## добавить ветку локальную в remote git

```bash
git push --set-upstream origin имя_ветки
```

## revert

Отменяет последний коммит(создает новый с отменой текущего):

```bash
git revert HEAD
```

Отменить несколько последних коммитов по одному:

```bash
# Выполнять по порядку, если есть конфликты - решать их после каждого
git revert HEAD~2
git revert HEAD~1
git revert HEAD
```

Отменить конкретный коммит(любой в истории)

```bash
git revert a21qwd4
```

Отменить пачку коммитов сразу

```bash
# Отменяем изменения, но не коммитим сразу
git revert --no-commit HEAD~2..HEAD

# проверяем что отменилось
git status

# фиксируем все одни коммитом
git commit -m 'revert: отмена последних 3-ех коммитов'
```
